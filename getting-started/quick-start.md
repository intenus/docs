# Quick Start

{% stepper %}
{% step %}
#### Connect Your Wallet

Connect your Sui-compatible wallet (Sui Wallet, Suiet, Ethos, etc.) to the Intenus interface.
{% endstep %}

{% step %}
#### Express Your Intent

Simply describe what you want to do:

* "Swap 1,000 SUI for USDC"
* "Lend 5,000 USDC at the best rate"
* "Set a limit order: buy USDC at $0.95 per SUI"
{% endstep %}

{% step %}
#### Review Options

Our AI will present you with the best execution strategies, showing:

* Expected output amount
* Estimated fees
* Execution time
* Risk assessment
{% endstep %}

{% step %}
#### Execute

Approve the transaction with one click. Intenus handles all the complex routing and execution automatically.
{% endstep %}

{% step %}
#### Track Results

Monitor your transaction in real-time and view detailed execution reports.
{% endstep %}
{% endstepper %}
